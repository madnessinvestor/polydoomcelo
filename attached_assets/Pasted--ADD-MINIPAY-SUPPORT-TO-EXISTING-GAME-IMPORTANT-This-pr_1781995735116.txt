---

## USER FLOW

### Mobile User

1. User installs MiniPay app.
2. User opens MiniPay.
3. User scans a QR Code displayed on the game website OR opens the game link directly from MiniPay.
4. MiniPay opens the game inside its built-in browser.
5. Wallet is detected automatically.
6. Wallet connects automatically.
7. User can play normally.
8. Wallet address becomes the player's unique ID.

No email.

No password.

No registration.

Wallet address is the account.

## WALLET DETECTION

Create:

```text
src/hooks/useMiniPay.ts
```

Responsibilities:

* Detect MiniPay
* Detect MetaMask
* Detect Celo-compatible wallets
* Auto-connect MiniPay
* Return wallet address
* Return connection status

Detection:

```javascript
window.ethereum?.isMiniPay
```

## CONNECTION RULES

If MiniPay detected:

* Connect automatically.
* Do not show Connect Wallet button.
* Load wallet address automatically.

If MiniPay not detected:
