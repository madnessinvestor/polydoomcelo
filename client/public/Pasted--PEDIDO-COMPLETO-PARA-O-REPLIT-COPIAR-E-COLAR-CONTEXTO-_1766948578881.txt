🔐 REGRA CRÍTICA – ACESSO PELA ARC NETWORK

O jogo SÓ PODE INICIAR se o jogador estiver conectado à ARC Network.

Implementação:

Antes de carregar o jogo, verificar:

window.arc || window.isArcConnected === true

Se NÃO estiver conectado:

Mostrar tela de bloqueio com texto:

“Conecte-se à ARC Network para jogar Gokuarc vs Criptoides”

Botão: Verificar conexão

O jogo NÃO inicia

Se estiver conectado:

Carregar o Phaser

Iniciar o jogo normalmente

Caso não exista SDK real, simular a conexão, mas deixar o código preparado para integração futura.
