🎮 PolyDoomArc

PolyDoomArc is a modern hybrid Web2 + Web3 game architecture built on the Arc Network, combining traditional backend technologies, decentralized smart contracts, and a cloud database to deliver a scalable, secure, and high-performance gaming experience.

This project demonstrates how real blockchain mechanics can coexist with practical off-chain infrastructure while leveraging the Arc ecosystem as its foundational layer.

🌐 Built on Arc Network

PolyDoomArc is natively designed for the Arc Network, taking advantage of its infrastructure for:

⚡ Efficient on-chain interactions

🔐 Secure smart contract execution

💱 Native compatibility with stable assets such as USDC and EURC

🧱 A scalable environment for real Web3 applications

🧪 Testnet-friendly experimentation and ecosystem development

The project architecture assumes Arc Network as the core settlement and logic layer for its decentralized components.

🔗 Web3 + Database Integration

PolyDoomArc combines:

🌐 Arc Network (smart contracts & on-chain logic)

🗄️ Supabase (PostgreSQL database)

⚙️ Node.js backend

🎮 Phaser + React frontend

to create a realistic foundation for a decentralized, scalable gaming ecosystem.

🗄️ Supabase (Off-Chain Database)

The project uses Supabase (PostgreSQL) as its primary off-chain database to store:

👤 Player profiles

🏆 Scores and match history

📊 Player statistics

🛒 Purchase records

⚙️ Settings and progression

This architecture provides:

High performance

Reliable persistence

Easy scalability

Low latency for gameplay features
