🎮 PolyDoom Celo

PolyDoom Celo is a modern hybrid Web2 + Web3 game architecture built on Celo Mainnet, combining traditional backend technologies, decentralized smart contracts, and a cloud database to deliver a scalable, secure, and high-performance gaming experience.

This project demonstrates how real blockchain mechanics can coexist with practical off-chain infrastructure while leveraging the Celo ecosystem as its foundational layer.

🌐 Built on Celo Mainnet

PolyDoom Celo is natively designed for Celo Mainnet, taking advantage of its infrastructure for:

⚡ Efficient on-chain interactions

🔐 Secure smart contract execution

💱 Native compatibility with stable assets such as USDC and cEUR

🧱 A scalable environment for real Web3 applications

📱 MiniPay and mobile-first wallet compatibility

The project architecture assumes Celo Mainnet as the core settlement and logic layer for its decentralized components.

🔗 Web3 + Database Integration

PolyDoom Celo combines:

🌐 Celo Mainnet (smart contracts & on-chain logic)

🗄️ PostgreSQL database

⚙️ Node.js backend

🎮 Phaser + React frontend

to create a realistic foundation for a decentralized, scalable gaming ecosystem.
