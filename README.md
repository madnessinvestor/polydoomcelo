🔗 Web3 + Database Integration

Polydoomarcunder combines traditional backend technologies with decentralized on-chain infrastructure and a cloud database to deliver a modern and scalable gaming experience.

🗄️ Supabase (Database)

The game uses Supabase (PostgreSQL) as its primary off-chain database to store:

👤 Player profiles

🏆 Scores and match history

📊 Player statistics

🛒 Purchase records

⚙️ Settings and progression

This architecture ensures high performance, reliable persistence, and easy scalability.

🧠 Hybrid Architecture (Web2 + Web3)

Polydoomarcunder follows a modern hybrid architecture:

⚡ Supabase (off-chain) → speed, performance, persistence

⛓️ Smart Contracts (on-chain) → transparency, security, decentralization

🎮 Phaser + React → smooth browser-based gameplay

⚙️ Node.js API → bridge between client, Supabase, and blockchain

This approach makes the project:
✔ scalable
✔ secure
✔ modern
✔ ready for real digital economy integration
